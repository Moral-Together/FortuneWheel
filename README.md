# 🎡 Колесо Фортуны

Интерактивное колесо фортуны в ярком детском стиле. Добавляй участников, крути колесо и узнавай победителя!

## Возможности

- Добавление и удаление участников
- Два режима игры: **Выбывание** (победитель исчезает) и **Повтор** (все остаются)
- Плавная анимация вращения с замедлением
- Звуковые эффекты (Web Audio API)
- Конфетти при победе
- История победителей текущей сессии
- Адаптивный дизайн (мобильный и десктоп)
- Данные хранятся в `sessionStorage` (сбрасываются при закрытии вкладки)

## Технологии

- React 18 + Vite
- Canvas API (рисование колеса)
- Web Audio API (звуки)
- Framer Motion (анимации UI)

## Запуск локально

```bash
npm install
npm run dev
```

## Деплой на GitHub Pages

```bash
npm run deploy
```
